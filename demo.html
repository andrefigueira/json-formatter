JAYSUN

A lightweight plugin that allows you to view JSON nicely.

Usage

Using JAYSUN is as simple as 1, 2, 3... Just follow the examples below or download the Github example.

JavaScript

$(document).ready(function(){
			
	$('.jFirmClick').jFirm({
		title: 'Hello World',
		content: 'Hello World This is a confirm.',
		cancel: 'Cancel',
		confirm: 'Confirm'
	});
	
});

HTML


<a href="javascript:{}" class="jFirmClick">Click me!</a>
