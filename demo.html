JAYSUN

A lightweight plugin that allows you to view JSON nicely.

Usage

Using JAYSUN is as simple as 1, 2, 3... Initiate the jQuery function poiting the function to the element containing the unformatted JSON string.

JavaScript

$(document).ready(function(){
			
	$('.json').Jaysun({
	  collapse: true, // Setting to 'true' this will format the JSON into a collapsable/expandable tree
	  closed:false //This only is valid if collapse is true, sets if the objects are visible or not
	});
	
});
